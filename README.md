## How To Play ##

| Controls                   |                   |
|----------------------------|-------------------|
| **LEFT** & **RIGHT ARROW** | move around       |
| **UP** & **DOWN ARROW**    | stretch / squeeze |
| **SPACEBAR**               | jump              |
| **X**                      | pick up / throw   |
